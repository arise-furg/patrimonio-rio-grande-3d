Como navegar

Use os botões de orientação para visualizar o modelo de frente, trás, esquerda, direita, topo e base. Para navegação livre, use o modo de rotação 360° e arraste com o mouse sobre o modelo.

Ferramentas do visualizador

A interface inclui rotação livre, pan, zoom, vistas ortogonais, recentralização do modelo e tela cheia.

Observação técnica

Caso o modelo não carregue, verifique se o arquivo .nxs foi enviado corretamente para a pasta models/.
